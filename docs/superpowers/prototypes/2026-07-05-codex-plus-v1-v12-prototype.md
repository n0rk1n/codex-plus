## 已确认界面规则

- 悬浮工作台是 V1 的优先入口。
- 整体视觉采用 macOS 26 液态玻璃风格。
- 不展示传统软件标题栏。
- 点击悬浮窗口外部只收起窗口，不终止后台任务。
- 右上角保留独立圆形大头钉按钮，用于固定悬浮窗口。
- 右侧区域完全收合，V1 主界面先不展示右侧信息栏。
- 左侧区域上移到顶部，以横向项目卡片展示。
- 顶部操作只保留 `新对话` 和 `已归档`。
- `搜索归档` 不作为独立按钮，后续放入 `已归档` 页面。
- 已归档会话不出现在活跃项目卡片或普通会话列表中。
- 每张顶部卡片使用两行区分项目和对话：
  - `项目图标 项目：项目名称`
  - `对话图标 对话：对话名称`
- 项目没有未归档会话时，对话行显示 `暂无对话`。
- 项目有一个未归档会话时，对话行显示该会话。
- 项目有多个未归档会话时，仍只展示一条，并显示计数下拉按钮，例如 `3 ▾`。
- 运行中的任务不允许发送新消息。
- 输入区右侧只保留一个主操作按钮。
- 任务运行中主按钮显示停止方块。
- 任务结束后同一个按钮切换为发送按钮。
- 停止和发送互斥，不同时展示。
- 底部状态栏只展示全局技术状态：
  - `Codex CLI 可用`
  - `SQLite 已连接`
  - `归档索引 待更新`
- 底部状态栏不展示 `悬浮窗 未固定`。
- 底部状态栏不展示 `任务 后台继续`。

## 仍需后续确认

- 多会话下拉展开后的列表样式和归档操作位置。
- 大头钉固定态的视觉状态。
